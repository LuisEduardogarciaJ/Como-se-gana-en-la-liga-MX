# Como-se-gana-en-la-liga-MX
¿Qué separa a los campeones del resto? Todo el análisis gira alrededor de eso.
Recopilé 4 capas de datos:

Historial de campeones (8 torneos, Apertura/Clausura 2021-2024)
Estadísticas acumuladas de los 18 equipos (PJ, PG, PE, PP, GF, GC, DG)
Métricas tácticas de campeones (efectividad, posesión, tiros al marco, errores defensivos)
Datos de inversión de plantilla (USD) para el análisis ROI
